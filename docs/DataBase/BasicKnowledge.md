关系模型中有三类完整性约束，分别是实体完整性约束、参照完整性约束和用户自定义完整性约束

使用存储过程通常具有以下好处：可增强SQL语言的功能和灵活性。良好的封装性。高性能。可减少网络流量。存储过程可作为一种安全机制来确保数据库的安全性和数据的完整性。

一个关系模式的所有属性集合是这个关系的主码或主键，则称这样的主码或主键为全码

数据聚集结构的一种有效方式是块结构方式

在存储过程中，用于为局部变量指定一个默认值的子句是default

活锁问题的解决与调度算法有关，一种最简单的方法是“先来先服务”

在MySQL数据库中，数据库系统对数据的安全管理是使用身份验证、数据库用户权限确认等访问控制措施

NoSQL是以互联网大数据应用为背景发展起来的分布式数据库管理系统 

数据库中存储的数据具有永久存储、有组织和可共享三个基本特点

数据控制语言（DCL）主要用于安全管理，例如确定哪些用户可以查看或修改数据库中的数据。数据控制语言主要包括的主要SQL语言如下
- GRANT：用于授予权限，可把语句许可或对象许可的权限授予其他用户和角色
- REVOKE：用于收回权限，其功能与GRANT相反，但不影响该用户或角色从其他角色中作为成员继承许可权限

数据模型通常由数据结构、数据操作和数据约束三个要素组成

按照运算符的不同，关系代数的操作可分为传统的集合运算与专门的关系运算

外模式对应到数据库中的概念就是视图

可以使用RENAME USER语句修改一个或多个已经存在的MySQL用户账号，其使用的语法格式是：
```sql
RENAME USER old_user TO new_user [,old_user TO new_user] ...
```

在数据库系统中，可串行性就是并发执行的正确性准则

MapReduce是一种简单易用的软件框架。基于它可以开发出运行在成千上万个结点上，并以容错的方式并行处理海量数据的算法和软件
